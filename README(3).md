# 🎓 Student Data Organizer

A simple **Python Console-Based Student Management System** that allows
users to add, display, update, and delete student records using a
menu-driven interface.

------------------------------------------------------------------------

## 📌 Project Overview

The **Student Data Organizer** helps manage student information
efficiently. It stores student details such as ID, name, age, grade,
date of birth, and subjects. The project demonstrates Python
fundamentals including lists, dictionaries, sets, loops, and
`match-case`.

------------------------------------------------------------------------

## ✨ Features

-   ➕ Add Student
-   📋 Display All Students
-   ✏️ Update Student Information
-   🗑️ Delete Student
-   📚 Display Subjects Offered
-   🚪 Exit Program

------------------------------------------------------------------------
